# Roadmap

This document outlines planned features and improvements for DocForge beyond the v1.0 release.

## Planned Features

### Batch Generation
- Generate multiple documents from a single template with different data sets
- CSV/Excel row-per-document mode
- Bulk export as zip archive

### Template Library
- Pre-built templates for common use cases
- Template sharing and import from URL
- Template versioning with diff view

### Real-Time Collaboration
- Multiple users editing the same document simultaneously
- Change tracking and conflict resolution
- User presence indicators

### Webhook Integrations
- Trigger generation from external events (CI/CD, form submissions)
- POST results to external endpoints on completion
- Scheduled generation via cron-like configuration

### PDF Output
- Direct PDF export alongside .docx
- PDF/A compliance for archival use cases
- Custom PDF styling options

### Custom Styling Themes
- Theme support for generated documents
- Corporate branding presets (colors, fonts, logos)
- Style override system for templates

### Advanced Data Transforms
- Pivot table transforms
- Cross-source joins (merge data from multiple files)
- Statistical aggregations (median, percentile, standard deviation)

### Template Validation
- Lint-style checks for template authoring issues
- Red text accessibility warnings
- Missing section structure detection

## Community Requests

This section will be updated based on community feedback and feature requests. Please open a [feature request](https://github.com/m-chimiste/docforge/issues/new?template=feature_request.md) to suggest new capabilities.
